# Use the Test Case Creation Tool to create test cases from a Lucidchart flowchart

Reminder: Make sure to follow the Shape and Design Standards while creating flows. Use this link to find the Shape and Design Standards for flows.

1. Install the TCCT onto your local system. For installation instructions, click here.
2. Open the TCCT and authenticate your application. This only needs to be done once. For authentication instructions, click here.
3. Open the target LucidChart.
4. Copy the entire URL or just the unique alphanumeric chart identifier. (Example: `https://lucid.app/lucidchart/<alphanumeric value>/edit?...`)
5. Paste the value in the `Chart Controls` text area.

6. Select the `Load Chart` button in the `Chart Controls` section.
7. If no errors are present, a dialog window will ask where you want to save the Excel spreadsheet. Warnings may still appear, but warnings do not prevent the creation of an output.
8. If errors are present, a dialog box will display with information regarding the errors. Correct the errors accordingly and repeat step 6.
